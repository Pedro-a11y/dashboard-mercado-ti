# Dashboard — Mercado de TI 2024

Dashboard interativo para análise do mercado global de tecnologia, com filtro por país e visualizações dinâmicas, construído com Streamlit e Plotly.

---

## Objetivo

Transformar os dados do Stack Overflow Developer Survey 2024 em uma aplicação web interativa, permitindo explorar linguagens mais usadas, modalidades de trabalho e distribuição salarial por país.

---

## Funcionalidades

- Filtro por país na barra lateral
- Top 10 linguagens mais usadas
- Modalidade de trabalho (remoto, híbrido, presencial)
- Distribuição salarial por país

---

## Tecnologias Utilizadas

- Python 3.x
- Streamlit
- Plotly
- Pandas

---

## Como Executar

1. Clone o repositório
2. Crie e ative o ambiente virtual: `python -m venv venv` e `venv\Scripts\activate`
3. Instale as dependências: `pip install pandas streamlit plotly`
4. Rode a aplicação: `streamlit run app.py`

---

## Dataset

- Fonte: [Stack Overflow Developer Survey 2024](https://survey.stackoverflow.co/2024)
- Respostas: 65.437 desenvolvedores
